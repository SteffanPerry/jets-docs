## Usage

    jets stacks [options]

## Description

List deployed stacks

## Options

```
-p, [--page=N]                             # Page number
    [--order=ORDER]                        # Order: asc or desc
                                           # Default: asc
    [--all-projects], [--no-all-projects]  # Show all stacks across all projects
                                           # Default: false
```
